# Jabi Consulting — Ivory edition
Anthropic-inspired ivory/coral redesign concept based on Hisham Jabi's design brief.
- Palette: ivory `#F5F1EA`, ink `#1A1A1A`, warm coral `#C96442`
- Typography: Instrument Serif (headings) + Inter (body)
- No carousels, no sliders, generous whitespace, subtle reveals only
